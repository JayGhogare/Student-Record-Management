# 🎓 Student Record Management System

A Python console-based application to manage student records using dictionaries.

## ✨ Features

- ➕ Add Student
- 📋 View All Students
- 🔍 Search Student
- ✏️ Update Student Marks
- ❌ Delete Student
- 🚪 Exit Program

## 🛠️ Technologies Used

- Python
- Dictionaries
- Functions
- Loops
- Conditional Statements

## 📚 Concepts Practiced

- Dictionary Operations
- CRUD Operations
- Functions
- User Input
- Menu-Driven Programming

## ▶️ How to Run

1. Clone this repository.
2. Open the project.
3. Run the Python file.
4. Follow the menu options.
